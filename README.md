# mypy-mode
Emacs compilation-mode based support for mypy-mode

## Requirements

Python 3.6 + due to Type Hint support expectations


## Usage


<!-- -----{install}-----<!--

M-x mypy-install  or pip install mypy

Add this to your init file:
(require 'sdev-mypy)

<!-- -------{Use}-------<!--

M-x sdev/py-type-check
